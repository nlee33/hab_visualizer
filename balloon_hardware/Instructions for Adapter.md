For the RS41 Adapter that utilizes a Schrader Valve, you need a tool to tap the threads for the valve.
  - Threading for Schrader valve is 1/8 in. NPT, so youll need to tap the adapter and then thread it in.
  - May want to add some type of sealant for the threads , ie. Loctite
  - Presta valve that has a Schrader adapter on it should work as well
