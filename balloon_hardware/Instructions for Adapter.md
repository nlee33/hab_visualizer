For the RS41 Adapter that utilizes a Schrader Valve, you need a tool to tap the threads for the valve.
  - A bike tool will work to tap the threads
  - Presta valve that has a Schrader adapter on it should work as well
  - May want to add some type of sealant for the threads , ie. Loctite

One hole is used to tie down the balloon to a weight, while preparing to fill, Use the other hole as the anchor point to your payload. Make sure that there is still someone holding onto the balloon as backup to make sure that the balloon doesnt fly away.
