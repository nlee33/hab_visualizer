How to use
1. Load a flightClick the file box or drag a CSV onto it. Raw Meshtastic exports and Grafana CSVs both work — format is auto-detected.
2. Multiple flightsAdd as many files as you like. Each gets its own colour. Use the eye button on each row to show/hide individual tracks.
3. ExploreClick any dot on a track to see altitude, speed and position. Drag the slider at the bottom to scrub through time.
4. PredictClick ⟳ Predict, select a loaded flight, enter wind data from a radiosonde sounding, and run the prediction to overlay a dashed predicted path.
Accepted columnsMeshtastic: latitude_i, longitude_i, altitude_hae, rx_time
Grafana: latitude, longitude, altitude, time
Sounding dataweather.uwyo.edu — station 72493 (Oakland, CA)
rucsoundings.noaa.gov
